What is Justitia

Justitia is a program that automatically checks the answers to programming exercises.

From the student's point of view, Justitia works like this:

  • The student writes a program for some exercise.
  • The student submits the program to Justitia.
  • Justitia compiles that program.
  • Justitia runs the compiled program, and compares the output to some reference output.
  • The student gets feedback on whether or not his program is correct.

About Justitia
